# 🗑️ 버전 코드 2의 APK 삭제 가이드

## 📍 현재 화면 위치

스크린샷을 보면:
- **"내부 테스트"** 페이지
- **"출시"** 탭이 선택되어 있음
- **"버전 코드"** 섹션에 **"3"**과 **"2"**가 파란색 링크로 표시됨

---

## ✅ 삭제 방법 (단계별)

### 방법 1: 버전 코드 링크 클릭 (가장 쉬움)

1. **"버전 코드"** 섹션에서 파란색 **"2"** 클릭
   - 화면 중앙에 "버전 코드" 아래에 "3"과 "2"가 보임
   - **"2"**를 클릭

2. 버전 코드 2의 상세 페이지로 이동

3. **"앱 번들 및 APK"** 섹션에서:
   - 버전 코드 2의 APK 파일 찾기
   - 파일 옆의 **"삭제"** 또는 **"제거"** 버튼 클릭
   - 확인 대화상자에서 **"삭제"** 확인

4. **"저장"** 또는 **"변경사항 저장"** 클릭

---

### 방법 2: "버전 수정" 버튼 사용

1. 화면 오른쪽 상단의 **"버전 수정"** (Edit version) 파란색 링크 클릭

2. 버전 편집 페이지로 이동

3. **"앱 번들 및 APK"** 섹션에서:
   - 버전 코드 2의 APK 파일 확인
   - 파일 옆의 **휴지통 아이콘** 또는 **"삭제"** 버튼 클릭

4. **"저장"** 클릭

---

### 방법 3: "앱 번들 탐색기" 사용

1. 왼쪽 사이드바에서:
   - **"테스트 및 출시"** → **"최신 버전 및 번들"** 클릭
   - 또는 상단의 **"앱 번들 탐색기"** 링크 클릭

2. **"앱 번들 탐색기"** 페이지에서:
   - 모든 업로드된 파일 목록 확인
   - 버전 코드 2의 APK 파일 찾기
   - 파일 옆의 **"삭제"** 버튼 클릭

3. 확인 대화상자에서 **"삭제"** 확인

---

## 🎯 추천 방법

**가장 쉬운 방법**: **방법 1** (버전 코드 "2" 클릭)

1. 현재 화면에서 **"버전 코드"** 아래의 파란색 **"2"** 클릭
2. 버전 코드 2 상세 페이지에서 APK 삭제
3. 저장

---

## 📸 화면에서 찾을 위치

### 현재 화면에서 보이는 것:
- ✅ **"버전 코드"** 텍스트
- ✅ 파란색 링크 **"3"** (클릭 가능)
- ✅ 파란색 링크 **"2"** (클릭 가능) ← **이것을 클릭!**
- ✅ 오른쪽 상단 **"버전 수정"** 링크

### 클릭 순서:
1. 파란색 **"2"** 클릭
2. 버전 코드 2 상세 페이지 열림
3. APK 파일 찾기
4. 삭제 버튼 클릭
5. 저장

---

## ⚠️ 주의사항

- 버전 코드 3의 AAB는 **절대 삭제하지 마세요!**
- 버전 코드 2의 APK만 삭제하세요
- 삭제 전에 확인 대화상자가 나타나면 **"삭제"** 또는 **"확인"** 클릭

---

## 🔍 찾기 어려운 경우

만약 위 방법으로 찾을 수 없다면:

1. **"버전 수정"** (오른쪽 상단 파란색 링크) 클릭
2. 편집 페이지에서 **"앱 번들 및 APK"** 섹션 확인
3. 버전 코드 2의 파일 찾기
4. 삭제

---

**가장 간단한 방법**: 화면 중앙의 **"버전 코드"** 아래에 있는 파란색 **"2"**를 클릭하세요! 🎯
